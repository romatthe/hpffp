* Notes
  
** GHCi commands:
=:type= or =:t= can be used to get a type of a value, expression, or function. For example:
#+BEGIN_SRC 
Prelude> :t 35
35 :: Num p => p
Prelude> :t "Hello World!"
"Hello World!" :: [Char]
Prelude> :t words
words :: String -> [String]
#+END_SRC

** Strings Vs Text
=String= is a very simple type. It is merely an alias for a linked list of =Char=\s, i.e. =[Char]=

#+END_SRC
